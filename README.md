## Needed Resources

- Docker

## Usage & Help

1. Download Docker Desktop (Windows/Mac) || Docker Engine (Linux)
2. Ensure Docker Service is running
3. Start up the backend, trackhounds-mariadb, and frontend containers
4. Navigate to http://localhost:3000
